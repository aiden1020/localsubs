LocalSubs

Setup

Real-time, fully local, free and open source subtitle translation for streaming video.

Checking local model

Looking for the native helper

1

Install the local runtime

Create the local environment used by the translation service. No subtitle text is sent to the cloud.

brew tap yourname/open-stream-subtitles
brew install open-stream-subtitles-helper
open-stream-subtitles-helper install-native-host
2

Prepare the model

Install or update the selected model channel. The current developer build detects the bundled local model.

python3 scripts/runtime/model_manager.py install --channel stable
Model

3

Start the model service

Install the native host once. Chrome starts the helper when subtitles need translation.

open-stream-subtitles-helper install-native-host
4

Use on streaming video

Open a supported streaming page with English subtitles enabled. The translated line appears near the original subtitle and can be dragged.
